# Travel Website

A travel website developed as a study and practice project using HTML and CSS.  
The goal is to create a **fully responsive** website for a travel agency, now focusing on the **desktop version**.

## Introduction

This project is part of **Module 1** of the **ElevateHer** program.  
The aim is to build a complete and responsive travel agency website using **only HTML and CSS**, without any frameworks.

The first phase focused on the mobile version. This stage now highlights the **implementation of the desktop version**, adapting the layout for larger screens with visual improvements, better content distribution, and enhanced usability.

## 🔗 Live Demo

[👉 Click here to view the website] https://travel-website-desktop-ig8k.onrender.com

## Features

- Responsive layout (**desktop-focused**)
- Main page with travel information
- Semantic structure using **HTML5**
- Modern styling with **CSS3**
- Optimized design for larger screens

## Technologies Used

- **HTML5**
- **CSS3**
