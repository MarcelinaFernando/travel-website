# Travel Website

A travel website developed as a study and practice project for HTML and CSS.  
The goal is to create a responsive website for a travel agency, starting with a mobile-first approach.

## Introduction

This project is part of Module 1 of the ElevateHer program.  
The objective is to build a responsive website for a travel agency using only HTML and CSS.  
The first phase of development focuses on creating the mobile version (mobile-version).

## Live Demo
[Click the link to view the website] https://travel-website-6l1o.onrender.com

## Features

- Responsive layout (Mobile First)
- Main page with travel information
- Semantic structure using HTML5
- Styling with CSS3
